# 🌟 Kelvin Atsu Djayouri — Portfolio

A premium, high-performance portfolio website architected for modern web standards. Built with **React 18**, **Vite**, and **Glassmorphism** design principles.

---

## 🚀 Key Features

- **💎 Sophisticated Aesthetics**: Implements a deep-glassmorphism design system with refined motion and vibrant color palettes.
- **⚡ Performance-First**: Optimized with Vite for near-instant load times and zero-lag interactions.
- **📱 Fully Responsive**: Seamlessly scales from iPhone 13 Pro viewports to ultra-wide 4K monitors.
- **🧠 Interactive Depth**:
  - **Dynamic Hero**: Real-time typing animation system highlighting core technical roles.
  - **UX Motion**: Scroll-driven entrance animations powered by custom React hooks.
  - **Skill Visualization**: Data-driven proficiency bars with staggered animation triggers.
  - **Contact Engine**: Production-ready form with field validation and instant toast feedback.

## 🛠️ Technical Architecture

- **Core**: [React 18](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS Modules (Zero-dependency, high performance)
- **Environment**: Configured for enterprise-grade Vercel deployment
- **SEO**: Complete Open Graph (OG) and Twitter card metadata implementation

## 📦 Local Development

1. **Clone the Source**:
   ```bash
   git clone https://github.com/devkad67/my_portfolio-.git
   cd my_portfolio-
   ```

2. **Setup Dependencies**:
   ```bash
   npm install
   ```

3. **Spin up Dev Server**:
   ```bash
   npm run dev
   ```

4. **Production Build**:
   ```bash
   npm run build
   ```
