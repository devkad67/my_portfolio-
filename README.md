# Kelvin Atsu Djayouri - Portfolio

A premium, modern, and responsive portfolio website built with React, Vite, and CSS Modules.

![Portfolio Preview](public/preview.png) *(Note: Add a real preview image to the public folder if available)*

## 🚀 Features

- **Glassmorphism Design**: Sleek, modern UI with frosted glass effects.
- **Micro-Animations**: Smooth entry animations and hover effects for an engaging UX.
- **Responsive Layout**: Optimized for all device sizes, from mobile to ultra-wide monitors.
- **Interactive Sections**:
  - **Dynamic Hero**: Typing animation highlighting core roles.
  - **Experience Timeline**: Vertical timeline of professional journey.
  - **Skill Bars**: Animated progress bars showing technical proficiency.
  - **Contact Form**: Validated contact form with instant feedback.
- **Fast Performance**: Built with Vite for rapid development and optimized builds.

## 🛠️ Tech Stack

- **Frontend**: React 18, Vite
- **Styling**: CSS Modules (Vanilla CSS)
- **Icons**: React Icons (Fa, Ri, Si, etc.)
- **Deployment**: Configured for Vercel

## 📦 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/devkad67/my_portfolio-.git
   cd my_portfolio-
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
