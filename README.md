# My Portfolio

A modern, fast, and responsive portfolio website built with [Astro](https://astro.build/).

## 🚀 Features

- **Fast by Default** - Built with Astro for near-instant page loads
- **Responsive Design** - Looks great on all devices
- **Modern Stack** - TypeScript support and optimized web components
- **Easy to Deploy** - Static site generation for reliable hosting

## 🛠️ Tech Stack

- **Framework**: [Astro](https://astro.build/) - The all-in-one web framework
- **Language**: TypeScript
- **Styling**: CSS

## 📋 Prerequisites

- Node.js (v16 or higher)
- npm or yarn package manager

## 🏁 Getting Started

### Installation

1. Clone the repository:
```bash
git clone https://github.com/devkad67/my_portfolio-.git
cd my_portfolio
```

2. Install dependencies:
```bash
npm install
```

### Development

Start the development server:
```bash
npm run dev
# or
npm start
```

The site will be available at `http://localhost:3000`

### Building

To build the site for production:
```bash
npm run build
```

The generated static files will be in the `dist/` directory.

### Preview

Preview the production build locally:
```bash
npm preview
```

## 📁 Project Structure

```
/
├── src/
│   ├── assets/        # Images and static assets
│   ├── layouts/       # Layout components (BaseLayout.astro)
│   ├── pages/         # Route pages (index.astro)
│   ├── styles/        # Global styles (global.css)
│   └── env.d.ts       # TypeScript definitions
├── astro.config.mjs   # Astro configuration
├── tsconfig.json      # TypeScript configuration
└── package.json       # Project dependencies
```

## 🔧 Configuration

The project is configured via:
- `astro.config.mjs` - Astro settings
- `tsconfig.json` - TypeScript settings
- `package.json` - Dependencies and scripts

## 🚀 Deployment

This Astro site can be deployed to various platforms:

- **Vercel** - Built-in Astro support
- **Netlify** - Zero-config deployment
- **GitHub Pages** - Simple static hosting
- **Any static web host** - Just upload the `dist/` folder
